# LigaDeportiva2

Aplicacion Angular de frontend para gestionar una liga deportiva escolar.

La app funciona con datos locales y guarda los cambios del usuario en `localStorage`, por lo que no necesita servidor ni base de datos para ejecutarse.

## Desarrollo

```bash
npm start
```

## Build

```bash
npm run build
```

## Tests

```bash
npm test
```
